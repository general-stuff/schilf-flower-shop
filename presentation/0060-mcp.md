# MCP Server

Add a `mcp` command to the CLI that starts an MCP server.

The MCP server must contain two tools (both backed by existing functions in the data access layer):

* List orders
* Place an order

If the CLI is used to start the MCP server, the server MUST NOT print any output to stdout. Log messages must be written to stderr.

Update the `flower-shop-cli` skill to describe the new tools. Include an example command line that shows how to call the MCP Inspector with the CLI server.
